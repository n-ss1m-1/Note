#### 1. 无重复字符的最长子串（LeetCode 3）✅

**题目**：
给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长子串** 的长度。

**核心思路**：
使用**不定长滑动窗口**，用哈希表记录窗口内字符的出现次数。右指针不断向右扩展窗口，当窗口内出现重复字符时，左指针向右收缩窗口，直到窗口内无重复字符，同时记录窗口的最大长度。
- **时间复杂度**：O (n)，每个字符最多被左右指针各访问一次
- **空间复杂度**：O (1)（字符集有限，如 ASCII 字符集）或 O (k)（k 为字符集大小）

```
class Solution {
public:
    //不定长滑动窗口
    int lengthOfLongestSubstring(string s)
    {
        unordered_map<char,int>window;
        int ans=0;
        int left=0;
        for(int right=0;right<s.size();right++)
        {
            char c=s[right];
            window[c]++;        //右入窗
            while(window[c]>1)  //左出窗（保证窗口内无重复字符）
            {
                window[s[left]]--;
                left++;
            }
            ans=max(ans,right-left+1);  //记录当前窗口长度
        }
        return ans;
    }
};
```

- **易错点**：
    - 哈希表记录的是**窗口内字符的计数**，而非字符的下标
    - 左指针需要持续右移，直到当前字符 `c` 的计数 ≤ 1，而非只移动一次
    - 窗口长度计算为 `right-left+1`，不要遗漏 `+1`

#### 2. 找到字符串中所有字母异位词（LeetCode 438）💡

**题目**：

给定两个字符串 `s` 和 `p`，找到 `s` 中所有是 `p` 的**字母异位词**的子串的起始索引，返回这些索引，答案顺序不限。

**核心思路**：
- 思路 1：**不定长滑动窗口（收缩法）**：右指针扩展，字符超标则移动左指针收缩，窗口长度达标时即为答案。
- 思路 2：**定长滑动窗口（双数组法）**：先初始化窗口，再整体滑动，每次移出左端、加入右端字符，直接比较两个计数数组是否相等。

##### 方法 1：不定长滑动窗口（收缩法）

```
class Solution {
public:
    // 不定长滑动窗口
    // 右指针进窗口 → 字符超标就移动左指针出窗口 → 窗口长度达标则记录
    vector<int> findAnagrams(string s, string p) 
    {
        int s_len=s.size(),p_len=p.size();
        vector<int>ans;
        vector<int>cnt(26);
        for(char c:p) cnt[c-'a']++;  // 统计p的字符

        int left=0;
        for(int right=0;right<s_len;right++)
        {
            char c=s[right];
            cnt[c-'a']--;           // 右端加入窗口
            
            while(cnt[c-'a']<0)     // 当前字符超标，收缩左边界
            {
                cnt[s[left]-'a']++;
                left++;
            }
            
            if(right-left+1 == p_len) // 窗口长度达标 = 异位词
                ans.push_back(left);
        }
        return ans;
    }
};
```

- **时间复杂度**：O(n)
- **空间复杂度**：O (1)（仅 26 字母数组）
- **优点**：通用滑动窗口模板，好理解、好记忆
- **易错点**：
    - 必须用 `while` 收缩左指针，不能用 `if`
    - 只有窗口长度 == p_len 时才记录答案

##### 方法 2：定长滑动窗口

```
class Solution {
public:
    // 定长滑动窗口（更快、更直观）
    // 先建初始窗口 → 整体滑动 → 每次删左加右 → 直接比较数组
    vector<int> findAnagrams(string s, string p) 
    {
        int s_len=s.size(),p_len=p.size();
        if(s_len < p_len) return {};
        
        vector<int> ans;
        vector<int> s_cnt(26);
        vector<int> p_cnt(26);
        
        // 初始化第一个窗口
        for(int i=0;i<p_len;i++)
        {
            s_cnt[s[i]-'a']++;
            p_cnt[p[i]-'a']++;
        }
        if(s_cnt == p_cnt) ans.push_back(0);

        // 滑动窗口：删除左端，加入右端
        // i表示即将移出窗口的字符下标
	    for(int i=0;i<s_len-p_len;i++) 
        {
            s_cnt[s[i]-'a']--;          // 移出窗口左端
            s_cnt[s[i+p_len]-'a']++;     // 加入窗口右端
            
            if(s_cnt == p_cnt)
                ans.push_back(i+1);
        }
        return ans;
    }
};
```

- **时间复杂度**：O(n)
- **空间复杂度**：O(1)
- **优点**：
    - 纯定长滑动，代码最短、最稳、最快
    - 直接比较两个数组，逻辑最清晰
- **记忆点**：
    - 先建窗 → 再滑动 → 删左加右 → 相等即答案

## 本题必记总结

1. **题型**：**定长滑动窗口 + 字母计数**
2. **最优写法**：**双数组定长滑动**（比赛 / 面试首选）
3. **关键判断**：两个字母计数数组完全相等 → 是异位词
4. **通用模板**：
    - 定长窗口：初始化 → 滑动（删左加右）→ 比较
    - 不定长窗口：右进 → 超标收缩左 → 长度达标记录

### 滑动窗口通用知识点总结

1. **核心思想**：用双指针 `left` 和 `right` 维护一个窗口，通过扩展右指针、收缩左指针来遍历所有合法窗口，在 O (n) 时间内解决子串 / 子数组问题。
2. **分类**：
    - **不定长窗口**：如 T3，窗口长度随条件动态变化，需满足 “无重复字符” 等约束
    - **定长窗口**：如 T438，窗口长度固定为 `p.size()`，只需在窗口长度达标时判断条件
    
3. **通用模板**：

    ```
    int left=0;
    for(int right=0;right<n;right++){
        // 右指针入窗：更新状态
        while(窗口不满足条件){
            // 左指针出窗：更新状态
            left++;
        }
        // 记录答案
    }
    ```

4. **引申知识点**：
    - 滑动窗口 vs 双指针：滑动窗口是双指针的一种特殊应用，专注于子串 / 子数组的连续区间问题
    - 定长窗口的优化：当窗口长度固定时，可直接用 `right-left+1 == len` 来判断是否达标，无需额外判断
    - 字符计数优化：小写字母场景优先用数组 `cnt[26]`，比 `unordered_map` 更快
