#### 1. 两数之和（LeetCode 1）✅

**题目**：
给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出 **和为目标值** _`target`_  的那 **两个** 整数，并返回它们的数组下标。
你可以假设每种输入只会对应一个答案，并且你不能使用两次相同的元素。
你可以按任意顺序返回答案。

**核心思路**：用哈希表存储「数值 - 下标」映射，遍历数组时检查 `target - 当前数` 是否存在于表中，存在则直接返回下标对，不存在则将当前数存入表。

- **时间复杂度**：O (n)，一次遍历 + 哈希表 O (1) 查找
- **空间复杂度**：O (n)，哈希表存储所有元素
- **代码亮点**：边遍历边查表，避免重复使用同一元素；`unordered_map` 保证高效查找
```
int n=nums.size();
unordered_map<int,int>val_idx;

for(int i=0;i<n;i++)
{
	int x=nums[i];
	auto it=val_idx.find(target-x);
	if(it!=val_idx.end()) return {it->second,i};
	val_idx[x]=i;
}

return {};
```
- **易错点**：注意返回顺序（先存的下标在前）、处理无结果情况（题目保证有解）

#### 2. 字母异位词分组（LeetCode 49）💡
**题目**：
给你一个字符串数组，请你将 字母异位词 组合在一起。可以按任意顺序返回结果列表。

**你的代码思路**：
- 方法 1（已实现）：将**字符串排序后作为哈希表的 key**，异位词排序后 key 相同，直接分组
```
unordered_map<string,vector<string>> str_vec;

for(auto& s:strs)
{
	string key=s;
	sort(key.begin(),key.end());
	str_vec[key].push_back(s);
}

vector<vector<string>>ans;
ans.reserve(str_vec.size());

for(auto& [_,v]:str_vec)
{
	ans.push_back(move(v));
}

return ans;
```
- 方法 2（备选）：**统计 26 个字母的出现次数**，拼接成唯一字符串作为 key，避免排序开销

```
	vector<vector<string>> ans;
	unordered_map<string,int> key_idx;
	
	for(string& s:strs)
	{
		int cnt[26]={0};
		for(char c:s) cnt[c-'a']++;
		
		string key;
		for(int i=0;i<26;i++) key+=to_string(cnt[i])+"# ";

		auto it=key_idx.find(key);
		if(it==key_idx.end())
		{
			key_idx[key]=ans.size();
			ans.push_back({s});
		}
		else
		{
			ans[it->second].push_back(s);
		}
	}
	return ans;
```
- **时间复杂度**：
    - 排序法：O (n * k log k)，n 为字符串数，k 为字符串最大长度
    - 计数法：O (n * k)，更高效（适合长字符串场景）
    
- **空间复杂度**：O (n * k)，存储所有字符串
- **优化建议**：
    - 用 `move(v)` 转移 vector 所有权，避免拷贝，提升性能
    - **计数法 key 拼接时用 `#` 分隔数字**（如 `2#1#0...`），**避免数字拼接歧义**（如 `12` 和 `1,2` 混淆）

#### 3. 最长连续序列（LeetCode 128）💡

**题目**：
给定一个未排序的整数数组 `nums` ，找出数字连续的最长序列（不要求序列元素在原数组中连续）的长度。
请你设计并实现时间复杂度为 `O(n)` 的算法解决此问题。

**核心思路**：用哈希集合去重，遍历每个数时，**只从序列起点（x-1 不存在于集合）开始**，向后查找连续数，记录最大长度。

- **时间复杂度**：O (n)，每个数最多被访问两次（一次遍历，一次作为起点向后查找）
- **空间复杂度**：O (n)，哈希集合存储所有元素
- **关键优化**：跳过非起点的数，避免重复计算，保证线性时间
- **代码模板**：
    ```
    unordered_set<int> st(nums.begin(), nums.end());
    int ans = 0;
    for (int x : st) {
        if (!st.count(x - 1)) { // 是序列起点
            int y = x + 1;
            while (st.count(y)) y++;
            ans = max(ans, y - x);
        }
    }
    return ans;
    ```
- **易错点**：
    - 必须先去重，否则重复元素会导致错误计数
    - 不要从每个数都向后遍历，否则时间复杂度退化为 O (n²)
