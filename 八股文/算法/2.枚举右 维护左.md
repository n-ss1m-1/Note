## 一、枚举右，维护左 

### 1. 核心思想

把**双变量问题**转化为**单变量问题**：

- 枚举右边的元素 `a[j]`
- 在 `a[j]` 左边的元素中，快速查找 / 统计满足条件的 `a[i]`（`i<j`）
- 用**哈希表 / 数组 / 前缀最值**维护左边信息，时间复杂度 O (n)

### 2. 通用模板

```
// 初始化：左边为空
unordered_map<int, int> cnt; // 或 vector<int> cnt(MAX_VAL, 0)
int ans = 0;

for (int j = 0; j < n; j++) {
    // 1. 用当前 a[j] 查询左边满足条件的数量/最值
    ans += cnt[target - a[j]]; // 或其他查询逻辑
    
    // 2. 把 a[j] 加入左边，供后续 j 使用
    cnt[a[j]]++;
}
```

### 3. 典型题目拆解

| 题目                         | 核心逻辑                                                        | 维护方式          |
| -------------------------- | ----------------------------------------------------------- | ------------- |
| **1. 两数之和**                | 找 `a[i] = target - a[j]`                                    | 哈希表存 `值→下标`   |
| **121. 买卖股票的最佳时机**         | 找 `j` 左边最小 `a[i]`，求 `a[j]-min`                              | 维护前缀最小值       |
| **219. 存在重复元素 II**         | 找 `j` 左边 `k` 范围内是否有 `a[j]`                                  | 哈希表存 `值→最近下标` |
| **1010. 总持续时间可被 60 整除的歌曲** | 找 `(time[i]+time[j])%60=0` → `time[i] ≡ (60-time[j]%60)%60` | 数组存余数计数       |

## 二、枚举中间 

### 1. 核心思想

针对**三 / 四变量问题**（如 `i<j<k`）：

- 枚举中间变量 `j`
- 左边 `i<j` 和右边 `k>j` 互相独立，分别维护信息
- 避免枚举 `i` 后还要处理 `j<k` 的复杂约束，时间复杂度 O (n)

### 2. 通用模板

```
// 预处理右边信息（后缀和/后缀最值）
vector<int> suf(n);
suf[n-1] = nums[n-1];
for (int i = n-2; i >= 0; i--) suf[i] = min(suf[i+1], nums[i]);

// 预处理左边信息（前缀和/前缀最值）
int preMin = nums[0];
int ans = INT_MAX;

for (int j = 1; j < n-1; j++) {
    // 1. 用当前 j 查询左边和右边信息
    if (preMin < nums[j] && nums[j] > suf[j+1]) {
        ans = min(ans, preMin + nums[j] + suf[j+1]);
    }
    // 2. 更新左边信息
    preMin = min(preMin, nums[j]);
}
```

表格

|题目|核心逻辑|维护方式|
|---|---|---|
|**2909. 元素和最小的山形三元组 II**|枚举 `j`，左边找最小 `a[i]<a[j]`，右边找最小 `a[k]<a[j]`|前缀最小值 + 后缀最小值|

## 三、遍历对角线 

### 1. 核心思想

矩阵中**同一条主对角线**的 `i-j` 为定值
矩阵中**同一条副对角线**的 `i+j` 为定值

对于主对角线：
- 用 `k = i-j + n` 标记对角线（`k` 从 `1` 到 `m+n-1`）
- 枚举每条对角线 `k`
	- `min_j` = max(n - k, 0)
	- `max_j` = min(m+n-1 - k, n-1)
	- 收集元素后排序 / 处理，再放回原位置
- 时间复杂度 `O (mn log (min (m,n)))`

### 2. 通用模板

```
int m = grid.size(), n = grid[0].size();
for (int k = 1; k <= m+n-1; k++) {
    int min_j = max(n - k, 0);
    int max_j = min(m+n-1 - k, n-1);
    vector<int> a;
    // 收集对角线元素
    for (int j = min_j; j <= max_j; j++) {
        a.push_back(grid[k+j-n][j]);
    }
    // 处理（排序/计算）
    sort(a.begin(), a.end());
    // 放回原位置
    for (int j = min_j; j <= max_j; j++) {
        grid[k+j-n][j] = a[j - min_j];
    }
}
```

### 3. 典型题目

- **3446. 按对角线进行矩阵排序**：左下三角（含主对角线）非递增排序，右上三角非递减排序，通过 `min_j>0` 判断分区后选择排序方向。

## 四、速记口诀 

1. **双变量 → 枚举右，维护左**：哈希表 / 数组存左边，O (n) 搞定。
2. **三变量 → 枚举中间**：左右独立，预处理前缀 / 后缀信息。
3. **矩阵对角线 → 用 k=i-j+n / k=i+j 标记**：收集 - 处理 - 放回，按对角线遍历。

---

## 五、易错点提醒 

- **枚举右时**：先查询再插入，避免把当前 `j` 自己算进去（保证 `i<j`）。
- **枚举中间时**：`j` 不能是首尾元素（需要至少一个 `i` 和一个 `k`）。
- **对角线遍历时**：注意 `i` 和 `j` 的坐标转换，避免越界。
