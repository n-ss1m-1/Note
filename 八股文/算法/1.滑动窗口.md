## 定长滑动窗口
##### 特征
- 窗口长度固定 k
##### 核心思路
1. 先算第一个窗口
2. 右边进一个，左边出一个，O (n) 走完
##### 典型题目
- 1456 定长子串中元音的最大数目
- 643 子数组最大平均数 I
- 1343 大小为 K 且平均值≥阈值的子数组数目
```
for(int right=0; right<n; right++)
{

	sum += nums[right];           //2.左进一个

	if(right < k-1) continue;     //1.先算第一个窗口

	max_sum = max(max_sum, sum);

	sum -= nums[right-k+1];       //3.右出一个

}
```

## 不定长滑动窗口

#### 求最长/最大
越短越合法，while 不合法缩左
###### 典型题目
- 3 无重复字符的最长子串
- 904 水果成篮
```
for(r=0; r<n; r++)
{
	cnts[fruits[r]]++;

	while(cnts.size() > 2)       //尽量短 越短越合法
	{
		--cnts[fruits[l]];

		if(cnts[fruits[l]] == 0) cnts.erase(fruits[l]);

		l++;
	}

	ans=max(ans, r-l+1);      //求最长/最大
}
```


#### 求最短/最小
越长越合法，while 合法缩左
###### 典型题目
- 209 长度最小的子数组
- 3795 不同元素和至少为 K 的最短子数组长度
```
for(r=0; r<n; r++)
{
	sum+=nums[r];

	while(sum-nums[l]>=target)     //预设减 在>=target的前提下 尽量减
	{
		sum-=nums[l];

		l++;
	}

	if(sum>=target) ans=min(ans,r-l+1);   //求最短/最小
}
```

#### 求子数组个数
- 越短越合法：ans += right-left+1
- 越长越合法：ans += left
- **恰好 K**：N(≥k) − N(≥k+1)
###### 典型题目
- 713 乘积小于 K 的子数组
- 1358 包含所有三种字符的子字符串数目
- 930 和相同的二元子数组
