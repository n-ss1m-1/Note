多表关系和多表查询
多表关系
	1.一对一：用于表结构拆分，在任意一方设置外键(unique)，关联另一方的主键
	2.一对多：在多的一方设置外键，关联一的一方的主键
	3.多对多：建立中间表，中间表包含两个外键，关联两张表的主键，且这两个外键作为联合主键(primary key(A,B))，允许A重复或B重复，但不允许重复的组合

多表查询
注意：
▲where -> on(连接条件的前标识)   
列子查询用in any all()  
行子查询用(字段列表)=()匹配
on后可以加where来进一步筛选
区分on的连接条件(受到连接方式影响) 和 where的筛选条件(不受连接方式影响)
可以连接多个表(多次join) 可以合并多个连接条件(and)
select  distinct去重
不太理解inner join 和 left join 的区别？：通过查询结果来区分

概述：
	1.多表查询消除笛卡尔积：使用on 相同的变量(=)匹配   通过外键匹配
	2.分类：
		2.1连接查询：▲当需要匹配两个表的内容时 需要使用连接(条件)
			2.1.1内连接：A∩B
			2.1.2外连接：
				2.1.2.1：左外连接：A∪(A∩B)
				2.1.2.2：右外连接：B∪(A∩B)
			2.1.3自连接：当前表与自身的连接查询，自连接必须使用表别名
		2.2子连接：
		

内连接：A∩B(▲查询结果：两者匹配成功的部分)
	语法：
		1.隐式内连接：select 字段列表 from   表1，表2   where 条件；
		2.显示内连接：select 字段列表 from  表1   inner join   表2   on 连接条件  ...；
	注意：1.关联的字段值为null时，将不会被查询到
		  2.起别名后，在该句中将不能再使用原名

外连接：(▲查询结果：左表保留全部字段。右表能匹配上的，显示对应字段；匹配不上的，右表字段全部填 NULL)
	语法：
		1.左外连接A∪(A∩B)：select 字段列表 from   表1 left outer join   表2  on 连接条件  ...；
		2.右外连接B∪(A∩B)：select 字段列表 from   表1 right outer join  表2  on 连接条件  ...；


自连接：看成两张表
	语法： 
		select 字段列表 from   表A 别名A  join  表A 别名B   on  条件  ...；
	注意：
		自连接查询，可以是内连接(inner join)，也可以是外连接(left join / right join)


联合查询：将多次查询的结果合并起来，形成一个新的查询结果集(可能有重复的行)
	语法：
		select 字段列表 from 表1 ...
		union [all]
		select 字段列表 from 表2...;
	注意：
		1.使用前提：字段列表必须保持一致(数量、类型)
		2.union all -> 可能会有重复的行
		   union      -> 不会有重复的行(类似于多表查询版的or)
		



子查询(嵌套查询)：
	概念：在SQL语句中嵌套select语句
		  子查询外部的语句可以是insert / update / delete / select 任意一个
		  例：select * from tb1 where column1 = (select column1 from tb2);
	分类：
	查询结果：
		1.标量子查询：子查询结果为单个值
		2.列子查询：子查询结果为一列 (可多行)
		3.行子查询：子查询结果为一行 (可多列)
		4.表子查询：子查询结果为多行多列
	子查询位置：
		1.where之后 
		2.from之后
		3.select之后

		
标量子查询：
	常用操作符：=   !=    >   >=   <   <=   <>   可以与and  or一起用
	例：查询晚于方东白入职的员工信息
	select * from emp where entrydate>(select entrydate from emp where name='方东白');

列子查询：
	常用操作符：in()   not in()   any()   some()   all()    可以与and  or一起用
	解释：
		1.in:寻找是否有匹配值   / not in相反
		2.in()   相当于   = any()
		3.any和some:与> < =等一起使用  列表中有任意一个满足即可
		4.all:与> < =等一起使用 列表中必须全部都满足     例： > any()
	
行子查询：
	常用操作符：=    !=    in()    not in()
	常用语法：... where  (字段列表)  操作符  (行子查询);
 	例：select * from emp where (salary,managerid) = (select salary,managerid from emp where name='张无忌');

表子查询：
	常用操作符：in()  any()  all()
	常用：
		1. ... where  (字段列表)  in  (表子查询);
		2.常放于from之后 作为一个临时表(必须起别名！！！) 与其他表进行联查操作
	注：即为列子查询(in any all->列匹配)+行子查询(多字段的列表->行匹配)
